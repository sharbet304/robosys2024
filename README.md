# robosys2024

ロボットシステム学の練習リポジトリ


# ohanaコマンドの概要

季節に合った花を教えてくれるプログラムです。 
季節と色を入力することで花の名前を教えてくれます。

# リポジトリのクローン方法

以下の手順を行って下さい。

```
git clone https://github.com/sharbet304/robosys2024
```

```
cd robosys2024
```

# 使い方

```
./ohana haru aka
```
